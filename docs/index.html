üõ†Ô∏è How to Use

Prerequisites

  • AI assistant with PDF reading capability (Warp AI, Claude, ChatGPT with PDF support)
  • A folder containing related PDF files
  • Basic command line access (optional)

Step-by-Step Guide

  1. Prepare Your PDFs

    Place all related PDFs in a single folder. Optionally name them in order (001_intro.pdf, 002_chapter1.pdf, etc.)

  2. Run the Basic Command
    In folder ./[YOUR_FOLDER]: create an interactive HTML doc that synthesizes the contents and provides a walkthrough guide. Ignore existing html/md files and roll up insights and pathways based on the PDF files in this folder.
  3. Customize (Optional)

    Add specific focus areas to your prompt:

    Create an interactive HTML guide from PDFs in ./documents/ focusing on: - Key decision points - Step-by-step processes - Common pitfalls and solutions - Best practices
  4. Review & Iterate

    The AI will generate an HTML file. Review it and ask for adjustments:

    Can you add a table of contents? Also make the timeline more visual with dates and milestones.

Advanced Customization

Specify structure:

Create an interactive guide from ./training/ organized by: 1. Overview and prerequisites 2. Core concepts with examples 3. Hands-on exercises 4. Troubleshooting FAQ 5. Additional resources

Target specific audience:

Synthesize these medical PDFs into a patient-friendly guide at a 6th grade reading level with visual diagrams and clear action steps.
